method generate_controller [line 166]

	

    

    

		void generate_controller(
string
$name, [string
$views = ""], [boolean
$scaffolding = false])
    

    


	
		Implement "generate controller" command

Example:
php script/generate.php controller   SomeName
   will generate:
  • a file
  app/controllers/some_name_controller.php
    
  containing the class definition
    
  class SomeNameController extends
  ApplicationController {}
  • a file
  app/helpers/some_name_helper.php
  • a directory
  app/views/some_name

  Optionally, one or more views can be appended to the command:
   php script/generate.php controller   SomeName view1 view2
   which will additionally generate files:
   app/views/some_name/view1.phtml
   app/views/some_name/view2.phtml






        
Parameters:

    

    

          

        string  
        $name  
        Name of the controller to generate in camel case
      

          

        string  
        $views  
        Optional list of views to generate
      

          

        boolean  
        $scaffolding
